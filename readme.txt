== Description ==

*   Output as feed the media library.

*   Outputs can be individually named.

*   Feeds of the image adds thumbnail.

*   Feeds of the audios and videos has been distributed as podcasts.


*   Japanese (ja) - [Katsushi Kawamori](http://gallerylink.nyanko.org/medialink/medialibrary-feeder/)

== Installation ==

1. Upload `medialibrary-feeder` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
